## 프로젝트명   
SimFarm 게임은 플레이어가 직접 동물들을 케어하면서 최고의 동물로 키워내는 게임입니다.   
## 프로젝트 설명   
SimFarm 프로젝트는 웹으로 게임을 만들어보고자 동물키우기 게임을 주제로 만들어본 프로젝트입니다.
